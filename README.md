# 유니티 프로젝트 
- 멀티플레이 가능한 유니티 게임을 제작하고 실제 배포까지 하는 것을 목적으로 하는 프로젝트입니다.
- 로그인 서버와 게임 서버를 두고서, 웹통신을 기반으로 동작합니다.
- 로그인 서버 및 게임 서버 - ASP.Net Core, 데이터 저장은 SQLite
- 로그인 웹사이트 - Next.js, github pages를 통한 배포. fetch를 통한 api 활용
- 게임 클라이언트 - Unity3D Engine. http와 gRPC를 활용한 서버 통신 
- 기술 스택: ASP.Net Core, Next.js, gRPC, SQLite, Unity, C#
- 자세한 설명은 블로그 글을 봐주세요 [링크](https://crusthack.github.io/blog/Project/Unity)
